# Política de Marca — Glória Finance

"Glória Finance" y sus logotipos son marca del proyecto.

## Uso permitido del nombre
Puedes usar el nombre "Gloria Finance" al distribuir el software **solo si**:

1. Mantienes avisos de copyright y atribución del proyecto.
2. Indicas claramente si tu versión es **oficial** o **modificada**.
3. Si es una versión modificada, debes:
    - publicar el código fuente de tus cambios (AGPLv3), y
    - describir los cambios principales en un archivo `CHANGES.md` o en la documentación.

## Uso no permitido
No puedes:
- Presentar un fork como si fuera la versión oficial.
- Usar el nombre o logo para confundir a usuarios sobre soporte, seguridad o propiedad.

## Permisos adicionales
Para usos comerciales, alianzas, white-label o excepción a estas reglas, contacta al mantenedor del proyecto.
